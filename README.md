## Run Locally

Clone the project

```bash
  git clone https://github.com/Fogha/netflix-clone.git
```

Go to the project directory

```bash
  cd Netflix-Clone
```

Install dependencies

```bash
  npm install
```

Start react app

```bash
  npm start
```
